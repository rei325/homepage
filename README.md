【成果物の説明文】
animejsを使っておみくじWebアプリを制作しました

＜工夫した点・ライブラリの活用箇所＞

1. おみくじ演出：
   「おみくじを引く」ボタンを押すと、Anime.jsを用いておみくじの箱がガタガタとリアルに振動し、その後にランダム抽出された運勢カードがイージング処理（バウンドするような弾む動き）を伴って画面に飛び出すアニメーションを実装しました。

2. 感想送信フォーム：
   おみくじ結果の下部に、名前とメッセージを入力できるフォームを設置しました。未入力状態で送信した場合はAnime.jsでエラー表示を揺らして強調し、正常送信時には感謝メッセージがふわっと浮き出る視覚的フィードバックを取り入れることで、ユーザーが心地よく操作できる工夫を行いました。
